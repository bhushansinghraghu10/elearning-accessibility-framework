# eLearning Accessibility QA Framework

## Project Overview
This repository contains a comprehensive manual testing framework focused on **Web Content Accessibility Guidelines (WCAG) 2.1**. It demonstrates a rigorous QA lifecycle for ensuring digital learning modules are accessible to all users, including those using assistive technologies.

## Key Deliverables
* **[Accessibility Defect Tracker (PDF)](https://github.com/bhushansinghraghu10/elearning-accessibility-framework/blob/main/Accessibility_Defect_Tracker.pdf):** A detailed log of accessibility barriers, categorized by severity and WCAG success criteria.
* **[WCAG QA Checklist](https://github.com/bhushansinghraghu10/elearning-accessibility-framework/blob/main/WCAG_QA_Checklist_32_Checkpoints.pdf):** A 32-point inspection list used to validate screen reader compatibility and keyboard navigation.
* **[QA Audit Report](https://github.com/bhushansinghraghu10/elearning-accessibility-framework/blob/main/QA_Audit_WSF_v2.1_B.Singh_FINAL.pdf):** A professional summary of the accessibility health of the "WSF eLearning Module."
* **[Module Draft Documentation](https://github.com/bhushansinghraghu10/elearning-accessibility-framework/blob/main/WSF_eLearning_Module_DRAFT.pdf):** The functional requirements and scope for the tested eLearning content.

## Skills Demonstrated
* **Accessibility Testing:** Screen reader validation and keyboard-only navigation.
* **Compliance:** Applying WCAG 2.1 Level A/AA standards.
* **Defect Management:** Professional bug reporting and tracking in Excel.
* **Documentation:** Creating clear, stakeholder-ready audit summaries.
