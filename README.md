# eLearning Accessibility QA Framework

## Project Overview
This repository contains a comprehensive manual testing framework focused on **Web Content Accessibility Guidelines (WCAG) 2.1**. It demonstrates a rigorous QA lifecycle for ensuring digital learning modules are accessible to all users, including those using assistive technologies.

## Key Deliverables
* **Accessibility Defect Tracker (Excel):** A detailed log of accessibility barriers, categorized by severity and WCAG success criteria.
* **WCAG QA Checklist:** A 32-point inspection list used to validate screen reader compatibility, keyboard navigation, and color contrast.
* **QA Audit Report:** A final professional summary of the accessibility health of the "WSF eLearning Module."
* **Module Draft Documentation:** The functional requirements and scope for the tested eLearning content.

## Skills Demonstrated
* **Accessibility Testing:** Screen reader validation and keyboard-only navigation.
* **Compliance:** Applying WCAG 2.1 Level A/AA standards.
* **Defect Management:** Professional bug reporting and tracking in Excel.
* **Documentation:** Creating clear, stakeholder-ready audit summaries.
